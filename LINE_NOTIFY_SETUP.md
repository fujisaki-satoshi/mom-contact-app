# 📲 LINE Notify トークン取得手順

このアプリを使うには、LINE Notifyのトークンが必要です。
以下の手順に従って取得してください。

## 📋 必要なもの

- LINEアカウント（通知を受け取りたいアカウント）
- スマートフォンまたはPC

## 🔑 トークン取得手順（画像付き解説）

### ステップ1: LINE Notifyにアクセス

1. **ブラウザで以下のURLを開く**
   ```
   https://notify-bot.line.me/
   ```

2. **「ログイン」をクリック**
   - 画面右上の「ログイン」ボタンをクリック

3. **LINEアカウントでログイン**
   - メールアドレスとパスワードを入力
   - または、QRコードでログイン

### ステップ2: マイページを開く

1. **右上のアカウント名をクリック**
   - ログイン後、画面右上に表示される自分の名前をクリック

2. **「マイページ」を選択**
   - ドロップダウンメニューから「マイページ」をクリック

### ステップ3: トークンを発行

1. **「トークンを発行する」ボタンをクリック**
   - マイページの下部にある青いボタンをクリック

2. **トークン情報を入力**

   **トークン名**（必須）
   ```
   お母様からの連絡
   ```
   - このトークンの用途を示す名前
   - 後で見てわかりやすい名前にする
   - 例: 「母の連絡アプリ」「かんたん連絡」など

   **通知を送信するトークルーム**（必須）
   - 以下のいずれかを選択:
     - ✅ **「1:1でLINE Notifyから通知を受け取る」**（推奨）
       - 自分だけに通知が届く
       - 他の人には見えない
     - 「グループを選択する」
       - 家族グループなど、複数人で共有したい場合

3. **「発行する」ボタンをクリック**
   - 内容を確認して、緑色の「発行する」ボタンをクリック

### ステップ4: トークンをコピー

1. **トークンが表示される**
   ```
   例: AbCdEfGhIjKlMnOpQrStUvWxYz1234567890
   ```
   - 長い英数字の文字列が表示されます

2. **⚠️ 重要: トークンをコピー**
   - 「コピー」ボタンをクリック
   - または、トークンを選択してコピー（Ctrl+C / Command+C）
   - **このトークンは一度しか表示されません！**
   - 必ずコピーしてください

3. **「閉じる」をクリック**
   - トークンをコピーしたら「閉じる」ボタンをクリック

### ステップ5: トークンを保存（推奨）

トークンを安全な場所に保存しておくことをお勧めします:

- **iPhoneのメモアプリ**
  - メモアプリを開く
  - 新規メモを作成
  - タイトル: 「LINE Notifyトークン」
  - トークンを貼り付け
  - メモをロック（セキュリティ強化）

- **パスワード管理アプリ**
  - 1Password、LastPassなど
  - 「LINE Notify トークン」として保存

## 📱 アプリでトークンを設定

### 初回設定

1. **アプリを開く**
   - iPhoneのSafariでアプリのURLを開く

2. **トークンを入力**
   - 設定画面が表示されます
   - 「LINE Notify トークン」欄をタップ
   - コピーしたトークンを貼り付け（長押し→ペースト）

3. **「保存して開始」をタップ**
   - トークンが保存されます
   - テストメッセージが送信されます

4. **LINEで確認**
   - LINEアプリを開く
   - 「LINE Notify」からメッセージが届いているか確認
   - 「✅ かんたん連絡アプリの設定が完了しました！」と表示されればOK

### トークンを変更する場合

1. **アプリの設定画面を開く**
   - アプリ下部の「⚙️ 設定」ボタンをタップ
   - 確認ダイアログで「OK」をタップ

2. **新しいトークンを入力**
   - 古いトークンは自動的に削除されます
   - 新しいトークンを入力
   - 「保存して開始」をタップ

## 🔒 セキュリティについて

### トークンの取り扱い

- ✅ **安全な保管**: トークンは他人に見せない
- ✅ **定期的な更新**: 必要に応じてトークンを再発行
- ✅ **不要になったら削除**: 使わなくなったトークンは削除

### トークンでできること

- ✅ **通知の送信**: 指定したトークルームにメッセージを送信
- ❌ **メッセージの読み取り**: 他のメッセージは読めません
- ❌ **アカウントの操作**: LINEアカウント自体は操作できません

### トークンを削除する方法

1. **LINE Notifyのマイページを開く**
   - https://notify-bot.line.me/my/

2. **「連携中のサービス」を確認**
   - 発行したトークンの一覧が表示されます

3. **削除したいトークンの「解除」をクリック**
   - 「お母様からの連絡」の右側にある「解除」ボタン
   - 確認ダイアログで「OK」をクリック

4. **トークンが無効化されます**
   - アプリからメッセージを送信できなくなります
   - 新しいトークンを発行して再設定が必要

## ❓ よくある質問

### Q1: トークンを忘れてしまった

**A:** トークンは再表示できません。新しいトークンを発行してください。

1. 古いトークンを削除（LINE Notifyのマイページ）
2. 新しいトークンを発行
3. アプリで新しいトークンを設定

### Q2: 複数のデバイスで同じトークンを使える？

**A:** はい、使えます。

- 同じトークンを複数のデバイスで設定可能
- すべてのデバイスから同じLINEアカウントに通知が届きます

### Q3: トークンが漏洩した場合は？

**A:** すぐにトークンを削除してください。

1. LINE Notifyのマイページでトークンを解除
2. 新しいトークンを発行
3. アプリで新しいトークンを設定

### Q4: 通知が届かない

**A:** 以下を確認してください:

1. **トークンが正しいか確認**
   - コピー時に余分なスペースが入っていないか
   - トークン全体がコピーされているか

2. **LINEの通知設定を確認**
   - LINEアプリの設定 → 通知
   - 「LINE Notify」の通知がオンになっているか

3. **トークンが有効か確認**
   - LINE Notifyのマイページで確認
   - 削除されていないか確認

4. **テストメッセージを送信**
   - アプリの設定画面で「保存して開始」をタップ
   - テストメッセージが届くか確認

### Q5: グループに通知を送りたい

**A:** トークン発行時にグループを選択してください。

1. トークン発行画面で「グループを選択する」を選択
2. 通知を送りたいグループを選択
3. グループのメンバー全員に通知が届きます
