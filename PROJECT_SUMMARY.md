# 📊 プロジェクトサマリー

## 🎯 プロジェクト概要

**プロジェクト名**: かんたん連絡アプリ  
**目的**: パーキンソン病で指先が震えるお母様でも簡単に連絡できるツール  
**対象ユーザー**: 文字入力が困難な高齢者とその家族  
**開発期間**: 2026年3月  
**バージョン**: 1.0.0

## ✨ 主な機能

### 1. 大きなボタンUI
- 画面の約1/4を占める大きなボタン
- 震える指でも押しやすい設計
- 色分けされた視認性の高いデザイン

### 2. 定型文メッセージ（8種類）
1. 🆘 助けてほしい（緊急）- 赤色
2. 📞 電話してほしい - 黄色
3. 😷 体調が悪い - オレンジ色
4. 😊 大丈夫です - 緑色
5. 💊 薬を飲みました - 青色
6. 🍚 ご飯を食べました - 青色
7. 💤 これから寝ます - 青色
8. 🙏 ありがとう - 紫色

### 3. LINE通知
- LINE Notify APIを使用
- 送信時刻付きでメッセージが届く
- 「【お母様から】」という接頭辞付き

### 4. PWA対応
- iPhoneのホーム画面に追加可能
- アプリのように使える
- オフライン対応（Service Worker）

## 🏗️ 技術スタック

### フロントエンド
- **HTML5**: セマンティックなマークアップ
- **CSS3**: グラデーション、レスポンシブデザイン
- **JavaScript (ES6+)**: モダンな構文、Fetch API

### PWA技術
- **Manifest.json**: アプリメタデータ
- **Service Worker**: オフライン対応、キャッシュ管理

### 外部API
- **LINE Notify API**: メッセージ送信

### ホスティング
- **GitHub Pages**: 無料の静的サイトホスティング
- **HTTPS**: 標準対応（PWAに必須）
