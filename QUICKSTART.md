# ⚡ クイックスタートガイド

このアプリを5分で始める方法です。

## 📝 3ステップで完了

### ステップ1: アプリを公開（3分）

1. **GitHubにアクセス**: https://github.com
2. **新規リポジトリ作成**: 右上の「+」→「New repository」
3. **ファイルをアップロード**: すべてのファイルをドラッグ&ドロップ
4. **GitHub Pages有効化**: Settings → Pages → Branch: main → Save

📌 詳しくは [DEPLOY.md](DEPLOY.md) を参照

### ステップ2: LINE Notifyトークン取得（1分）

1. **LINE Notifyにアクセス**: https://notify-bot.line.me/
2. **ログイン**: LINEアカウントでログイン
3. **トークン発行**: マイページ → トークンを発行する
4. **トークンをコピー**: 表示されたトークンをコピー

📌 詳しくは [LINE_NOTIFY_SETUP.md](LINE_NOTIFY_SETUP.md) を参照

### ステップ3: アプリを設定（1分）

1. **アプリを開く**: iPhoneのSafariでGitHub PagesのURLを開く
2. **トークンを入力**: コピーしたトークンを貼り付け
3. **保存**: 「保存して開始」をタップ
4. **ホーム画面に追加**: Safari下部の共有ボタン → ホーム画面に追加

## ✅ 完了！

これで使えるようになりました。
ホーム画面のアイコンをタップして、ボタンを押してみてください。

## 🎯 使い方

1. ホーム画面のアイコンをタップ
2. 送りたいメッセージのボタンをタップ
3. 「送信しました！」と表示されたら完了
4. 受信者のLINEに通知が届きます

## 📱 定型文の例

- 🆘 助けてほしい（緊急）

- 😷 体調が悪い
- 😊 大丈夫です
